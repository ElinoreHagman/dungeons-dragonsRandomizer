Project description

This is a Dungeons & Dragons randomizer where a character sheet is created. Spice it up and try to manage a fully randomized character in a one-shot campaign, or get basic ideas and change stuff afterwards. The randomizer is only covering what the official 5th Edition SRD has distributed, so only for example the basic classes and races are avaliable.

How it works

When randomizing a character the app fetches information from the dnd5eapi.co using jQuery, and picks out random objects using the Math.Random() function. Text regarding the backgrounds are picked from github.com/rushl, but used with my own randomizer functions.
