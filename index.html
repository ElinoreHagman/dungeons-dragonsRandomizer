Project description

This is a Dungeons & Dragons randomizer where a character is created for you to use in a campaign. The randomizer is only covering what the official 5th Edition SRD allows, so only for example the basic classes and races are avaliable.

How it works

When randomizing a character the app fetches information from the dnd5eapi.co using JQUERY, and picks out a random object using the Math.Random() function. Subraces and subclasses have a 40% chance of being chosen. Backgrounds are picked from a generator at github.com/rushl with some code tweaking to fit this app.
